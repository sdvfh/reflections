Quando gostaria de usar um repositório remoto em vez de manter o trabalho local?
R - Quando eu estiver trabalhando com um grupo.

Por que você prefere efetuar pull de alterações manualmente do que deixar o Git atualizar-se automaticamente com o seu repositório remoto?
R - Para eu ter o controle do que baixar ou não. Em outras palavras, do que ter acesso no 
local de trabalho.

Descreva as diferenças entre bifurcações, clones e branches. Quando você usaria um e não os outros?
R - Clone você baixa, branch você cria/nomeia um commit e bifurcação é mais a nível do github.
Bifurcação é para copiar o repositório de alguém para seu próprio github. Clone é para fazer
alterações no computador e branch para salvar essas alterações de acordo com o gosto do
usuário.

Qual é o benefício de ter uma cópia do último estado conhecido do remoto armazenado localmente?
R - De saber as alterações feitas por outros usuários e comparar com a sua versão armazenada localmente.

Como você colaboraria sem usar Git ou GitHub? O que seria mais fácil e mais difícil?
R - Utilizando o google drive e compartilhando os arquivos. Praticamente tudo que tem no google drive ou outro aplicativo que eu usasse
também há uma ferramenta semelhante no git, ou seja, nada seria mais fácil; pelo contrário, seria mais difícil sem o git.
